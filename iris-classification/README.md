# Iris Flower Classification (scikit-learn)

Klasik **Iris** veri seti ile makine öğrenmesine giriş örneği.
- Modeller: Logistic Regression, Decision Tree
- Metrik: Accuracy
- Grafikler: Özellik histogramları, karışıklık matrisi
